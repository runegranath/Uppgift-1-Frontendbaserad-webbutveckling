Hem Bilder Process

Välkommen!

Syftet med denna webbplats är att bygga något enkelt för att sedan automatisera utvecklingsmiljön.

Verktygen som automatiserar hjälper till med att:

  • Minifiera CSS och JavaScript
  • Optimera bilder
  • Kopiera filer till en publiceringsmapp
  • Publicera webbplatsen på ett lämpligt sätt

I uppgiften användes Vite som byggverktyg för automatiseringen.
