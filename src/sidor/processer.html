Hem Bilder Process

Processer

I denna uppgift har vi utöver HTML, CSS och javascript använt NodeJS och Vite som byggverktyg för att automatisera utvecklingsmiljön. Arbetet började med att använda npm init i terminalen för att skapa en .json-fil som kan konfigureras, där ändrade jag till att Vite skulle användas under script. När sedan jag använder kommandot npm run dev i terminalen startas en utvecklingsserver där ändringar som görs direkt uppdateras i webbläsarfönstret.

För att sedan lägga till Sharp som bildoptimerare i projektet användes ett Sharp-plugin till Vite genom att skriva npm install vite-plugin-imagemin sharp --save-dev.

Git har använts för att versionshantera koden och node_modules-mappen är exkluderad i publiceringen för att undvika onödiga filer laddas upp.

För att javascript skulle följa med i publiceringen och optimeras med vite behövde "script type="module"" läggas till i HTML.

Innan publiceringen installerade jag GitHub Actions extension och skapade en mapp som hette .github/workflows/ och i den en fil deploy.yml med instruktioner hur den ska automatisera publiceringen till GitHub Pages.

För att optimeringen av bilder skulle fungera behövdes inställningarna för vites plugin vara konfigurerade i vite.config.js. Där specificerades också "rollupOptions" för att få med fler undersidor än bara index.html.

När jag sedan ville publicera webbplatsen la jag till alla filer, committade och pushade via git efter att ha mergat branches mellan development och main som då själv kör npm install och npm build. Detta skapar en mapp med namnet "dist" som innehåller alla filer som behövs för att publicera webbplatsen. All CSS, alla bilder och Javascript blir då också optimerade.
